# 定义类加载器

如果某个类加载器能够加载一个类，那么该类加载器称为：定义类加载器。

定义类加载器及其所有子类加载器称为：初始类加载器。

当生成一个自定义类加载器的实例时，如果没有指定它的父类加载器，那么系统类加载器就成为该类加载器的父加载器。
