## Paralle Desktop 新建虚拟机

注意一下：Paralle 安装 CentOS7 有两个坑

1、要选择自动安装，勾选安装前设定

2、安装前设定，要设置虚拟机监控程序为 Paralle，默认是 Apple

第二点非常重要，要不然启动会失败。
