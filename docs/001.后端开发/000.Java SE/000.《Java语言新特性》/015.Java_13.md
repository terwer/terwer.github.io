JDK 13 于 2019 年 9 月 17 日发布。

## 主要特性

该版本包含以下新特性以及很多的强化和很多的漏洞修补。

- JEP-350: 动态 CDS 归档
- JEP-351: ZGC: 取消提交未使用存储器
- JEP-353: 重新实现旧版 Socket API
- JEP-354: Switch 表达式（预览阶段）
- JEP-355: 文本框（预览阶段）
