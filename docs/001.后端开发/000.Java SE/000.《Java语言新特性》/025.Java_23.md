Java 23 于2024年9月17日发布，引入了12项JEP更新，包括模式匹配预览、Vector API孵化和模块导入等新特性。文章提供了Oracle官方JDK和Adoptium开源版本的下载链接，并指出字符串模板功能被移除。

<!-- more -->






Java 23 在 2024 年 9 月 17 日发布，其 JEP 如下：

- JEP 455：模式中的原始类型、instanceof 和 switch（预览）
- JEP 466：类别文件 API（第二个预览版）
- JEP 467：Markdown 文件注释
- JEP 469：Vector API（第八个孵化器）
- JEP 473：流收集器（第二预览版）
- JEP 471：弃用 sun.misc.Unsafe 中的存储器访问方法并将其删除
- JEP 474：ZGC：默认的分代模式
- JEP 476：模块导入声明（预览）
- JEP 477：隐式宣告的类别和实例主要方法（第三次预览）
- JEP 480：结构化并发（第三次预览）
- JEP 481：作用域值（第三次预览）
- JEP 482：灵活的构造函数主体（第二次预览）

由于功能设计方面的问题，字符串模板预览功能在 Java 23 中被移除。
