# Deepin搭建GO开发环境

## 安装

下载

[https://go.dev/dl/](https://go.dev/dl/)

```bash
wget https://go.dev/dl/go1.18.3.linux-amd64.tar.gz
```

解压

```bash
sudo tar -C /usr/local -xzf go1.18.3.linux-amd64.tar.gz
```

设置环境变量

```bash
# GO
export GO_HOME=/usr/local/go
export PATH=$PATH:$GO_HOME/bin
```

验证

```go
go version
```
