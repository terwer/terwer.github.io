在 Linux 中，利用 bash 获取文件名和后缀还是很常见的。

## 获取后缀名

```bash
ext = ${file: -4}
echo $ext
```

如果文件名是 `file.zip`，运行结果如下：

```bash
.zip
```

## 获取文件名

```bash
name = $(basename "$file" .zip)
echo $name
```

如果文件名是 `file.zip`，运行结果如下：

```bash
file
```
