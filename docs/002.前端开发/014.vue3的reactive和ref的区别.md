# vue3的reactive和ref的区别



首先我们都知道 ref 函数和 reactive 函数都是用来定义响应式数据；

但是 reactive 更适合定义复杂的数据类型（json/arr）、ref 适合定义基本数据类型（可接收基本数据类型和对象）

一句话：

使用场景，ref 简单数据，reactive 复杂数据，ref 取数据需要加 value
