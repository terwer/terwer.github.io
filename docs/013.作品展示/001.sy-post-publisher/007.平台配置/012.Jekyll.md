## 安装 Jekyll

```bash
gem install jekyll -- --with-cppflags=-I$(brew --prefix openssl)/include
```

## 查看 Jekyll 版本

```bash
jekyll -v
```

## 站点初始化

```bash

git rm -rf .
```

## 创建新站点

```bash
jekyll new --skip-bundle .
```

## 替换依赖

```bash




gem "github-pages", "~> 227"
```

## 安装依赖

```bash
bundle install
```

解决依赖问题

```bash
bundle add webrick
```

## 本地运行

```bash
bundle exec jekyll serve
```
