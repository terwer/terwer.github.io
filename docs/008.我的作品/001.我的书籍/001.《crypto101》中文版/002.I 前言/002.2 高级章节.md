# 2 高级章节

这本书旨在成为程序员实用的密码学指南。一些部分深入探讨的程度超出了实现这一目标的必要程度。它们仍然包含在书中，以防你有兴趣；但我通常建议跳过这些部分。它们会以这样的方式标记：

> 这是一个可选的、深入的部分。它几乎肯定不会帮助你更好地编写软件，所以请随意跳过。它只是在这里满足你内心极客好奇的需求。
